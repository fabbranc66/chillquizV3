# StemPad

Progetto separato per la skin VirtualDJ `StemPad`, basato sul layout mostrato nello screenshot condiviso.

## Struttura

- `source/`: file XML della skin e varianti.
- `preview/`: mockup HTML/CSS locale per visualizzare il layout.
- `notes/`: appunti tecnici, mappa controlli e fix da fare.

## Stato

- Progetto inizializzato.
- Creato un mockup locale del layout per lavorare con piu precisione.
- Aggiunta una mappa dei controlli ricavata dallo screenshot.

## Flusso consigliato

1. usare il mockup in `preview/` come riferimento visivo;
2. completare `source/StemPad.xml` con il markup reale;
3. allineare posizioni, dimensioni e azioni VirtualDJ;
4. fare una pulizia finale della sintassi.
